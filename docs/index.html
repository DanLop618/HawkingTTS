Introduction

Hawking's is a Google Cloud Text to Speech API based Discord bot. But it is not just a simple bot. It has tons of features and customizations that will make it fit the needs of each server without losing the fun.

Tutorial

This tutorial is in Spanish only

Modules:

Hawking's works based on modules. These modules determine the behavior of the bot within the servers.
These modules are:

  • Text to Speech
  • Voice Utilities
  • Temporary Voice Channels  (under development) 
  • Speech to Text  (under development) 

Main Features:

Hawking's main features are:

  • Text to Speech
  • Customizable Voice Roles
  • Customizable Reading Voices
  • Custom Commands
  • Acronyms Dictionaries

Initial Setup

Inviting Hawking's

First of all, you need to Invite Hawking's to your Discord Server. After that, a message should be recieved in one of your server's text channels. Take a look around!

Server Setup

If any message is recieved when Hawking's joins your server, then... dont worry! Just do the following:

  •  /server_language 
    • Choose your server's default language.
    This language will be used for read all the users' messages with the Text to Speech module. But also, this will be used for the bot embedded replies.

  •  /whitelist 
    • Whitelist some channels.
    The bot itself can only read those messages sent in whitelisted channels. Make sure you whitelist at least one channel for the bot to work.

  •  /modules 
    • Choose the modules you want to be active.
    Some people just want Hawking's to be a text to speech bot... And that's ok! That's the main function for which the bot was made in first place. Enable and disable the modules you want. It is your choice!
